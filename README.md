**HTTP adapter integration tests.**


## Install

Via Composer

``` bash
$ composer require php-http/adapter-integration-tests
```


## Usage

This package should not be used on its own. It provides integration tests for HTTP Adapters.


## Testing

``` bash
$ phpunit
```
